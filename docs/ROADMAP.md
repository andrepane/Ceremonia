# Roadmap por fases

## Fase 1 · Demo sólida (actual)
- Flujo idioma → invitado → app con navegación funcional.
- Textos externalizados por idioma (ES/IT).
- Datos demo centralizados para invitados, ranking, retos y fotos.

## Fase 2 · Beta privada
- Conectar API para invitados reales y sesiones.
- Traductor real (texto a texto + audio).
- Subida de fotos real con moderación básica.

## Fase 3 · Producción boda
- QA final en dispositivos iOS/Android.
- Modo offline básico para zonas sin cobertura.
- Panel simple para activar/desactivar módulos durante el evento.
